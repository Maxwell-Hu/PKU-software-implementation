Description:
本应用主要面向中小学教育，为学生提供实时的智能问答服务，针对低年级同学的人群特点，提供精准回答、智能纠错、疑难问题收集、闲聊等功能，经测试，系统各个模块的算法评价指标均已达到业界标准，可以投入应用。
